# OS ダッシュボード

Read Not Store 原則のクエリビュー。ここは情報を保存せず、開くたびに Vault の最新状態を Dataview で都度生成する。**閲覧モード（リーディングビュー）**または Live Preview で開くこと。Dataview プラグインが必要（導入済み）。

このノートをブックマーク（スター）しておくと、毎朝の起点になる。

---

## 今日の日次ノート

```dataview
LIST
FROM "01_Daily"
SORT file.name DESC
LIMIT 1
```

## 期限が近い / 超過しているタスク

```dataview
TABLE WITHOUT ID file.link AS "タスク", priority AS "優先度", deadline AS "期限", file.folder AS "状態"
FROM "05_Tasks"
WHERE deadline AND !contains(file.folder, "done")
SORT deadline ASC
```

## オープンループ（未完了チェックボックス）

```dataview
TASK
FROM "01_Daily" OR "03_Projects"
WHERE !completed
SORT file.name DESC
```

## 00-CAPTURE 未処理（02_Inbox）

```dataview
LIST
FROM "02_Inbox"
SORT file.mtime DESC
```

## 05-QUEUE タスクキュー

`inbox`＝投入待ち / `active`＝実行中 / `blocked`＝あなたの判断待ち

```dataview
TABLE WITHOUT ID file.link AS "タスク", priority AS "優先度", delegate AS "担当", file.folder AS "状態"
FROM "05_Tasks/inbox" OR "05_Tasks/active" OR "05_Tasks/blocked"
SORT file.folder ASC, priority ASC
```

## 進行中プロジェクト

```dataview
TABLE WITHOUT ID file.link AS "プロジェクト", status AS "状態", owner AS "担当", file.mtime AS "最終更新"
FROM "03_Projects"
WHERE type = "project" AND status = "active"
SORT file.mtime DESC
```

## OS生成物（07_Generated 最新10件）

```dataview
TABLE WITHOUT ID file.link AS "生成物", file.folder AS "種別", file.mtime AS "生成日時"
FROM "07_Generated"
WHERE file.name != "README"
SORT file.mtime DESC
LIMIT 10
```
