# digests — デイリーダイジェスト

国際関係 / AI / アニメ・IP の3ジャンルの注目トピックを、毎朝7:00に自動生成する仕組みの出力先です。

## 仕組み

launchd ジョブ（Cowork 非依存）が毎朝 `daily-digest.py` を発火 → Grok（xAI API、未設定時は Claude Code）が3ジャンルを調査 → `YYYY-MM-DD.md` としてここに保存されます。

## フィードバックループ

各ダイジェストの末尾に「## フィードバック」欄があります。そこに感想や要望を書いてください（例:「AIは研究寄りで」「国際関係は中東を厚く」）。翌朝の生成時に、その内容が [[_preferences]] の履歴へ追記され、重みづけに反映されます。長期的な好みは [[_preferences]] の「恒久的な指示」欄に直接書けます。

## ファイル

- `YYYY-MM-DD.md` — その日のダイジェスト（システム生成。フィードバック欄のみ記入可）
- `_preferences.md` — 嗜好メモ。フィードバックの蓄積＋あなたの恒久指示
