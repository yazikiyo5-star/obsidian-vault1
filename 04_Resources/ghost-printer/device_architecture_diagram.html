Ghost-Printer デバイス構成図

指にはめた小さなリングが、あなたの身体のデータを集めて、Ghost-Printer本体に無線で送ります。本体のなかではAIがそのデータを解釈し、あなた専用の「SOUL（魂のデータ）」を生み出します。2026-04-18時点の構成です。

01全体像

2つのデバイスが無線でつながっています。左：リング（身体を測るセンサー）、右：Ghost-Printer本体（考えて出力する頭脳）。その間は BLE（Bluetooth Low Energy） という省電力無線でデータが流れます。

リング（入力デバイス）
無線通信
本体（処理デバイス）
補助的な入力
電源

02部品の名前と役割

それぞれのパーツが何で、何の役割を果たすかを、日常の比喩を添えて説明します。

スマートリング側

COLMI R02

指にはめる小さな指輪型のセンサー。常時身につけて身体の状態を測り続けます。価格は約3,000円。

  • PPGセンサー
    緑のLEDを指にあてて、血流の変化から心拍数と血中酸素（SpO2）を測る。 〝ドラッグストアでおなじみの指クリップ式パルスオキシメーター〟を極小にしたもの
  • 加速度センサー
    STK8321
    指の動きを3方向で検知して、歩数や活動量を推定する。 〝スマホを振ると反応する機能〟と同じ仕組み
  • BLEチップ
    BlueX RF03
    リングの頭脳。センサーの値をまとめて無線でGhost-Printerに送る。 〝小さなトランシーバー〟
  • バッテリー
    17mAh
    リングの電源。4〜5日に1回、専用ドックで充電する。 イヤホン用小型バッテリーの1/10程度
  • 防水
    5ATM
    水深50m相当まで耐える。手洗い・入浴・軽いプール程度ならつけたまま。

Ghost-Printer 本体側

Radxa Zero 3W

65×30mmの手のひらサイズのミニコンピュータ。リングからデータを受け取り、ローカルAI（Bonsai）で意味抽出・重要度判定を行い、SOUL（魂データ）に組み上げます。※ SOULを使って実際に会話するのは外部AI（Claude等）の役割。

  • メインボード
    Radxa Zero 3W
    脳と心臓を兼ねた基板。4GBのメモリで意味抽出AI（Bonsai 1.7B）を動かせる。会話生成は行わず、データをSOULに変換する役目。 〝Raspberry Pi Zeroの中華版（より速い）〟
  • SoC
    Rockchip RK3566
    ボードの中心。CPU × 4コア + NPU（AI専用の計算回路）を持つ1つの大きなチップ。 〝人の左脳（CPU）＋直感処理（NPU）〟
  • BTチップ
    AP6212A系
    リングから飛んでくるBLE信号を受信する耳。ボードに最初から内蔵されている。 〝ワイヤレスイヤホンの受信部〟
  • microSD
    32GB A2
    OS（Debian Linux）とデータを保存するカード。差し替えで環境を作り直せる。 〝脳の長期記憶〟
  • マイク
    ICS-43434
    音声・環境音を拾う入力。リングでは取れない「声」を補う。 〝耳〟
  • バッテリー
    LiPo 500mAh
    本体の電源。リチウムポリマー、3.7V。 〝ワイヤレスイヤホンのケース程度〟
  • Pogo Pin
    4極
    Soul Dock（将来の土台）と接続する金色のピン。充電やデータ転送に使う。 〝Apple Watchの背面の磁気充電部〟

03データの流れ

あなたの身体の情報が「数字 → 状態 → 意味 → SOUL → 外部AIの返答」と姿を変えていく旅路です。Bonsaiは意味抽出、会話生成は外部AIが担当します。

04用語集

このページで出てきた専門用語を、一言で説明します。

BLE
Bluetooth Low Energy。電池を食わない近距離無線。スマートウォッチやワイヤレスイヤホンと同じ仕組み。
PPG
光電脈波。緑のLEDを肌にあて、血流の変化を光で読み取る技術。心拍計測に使う。
HRV
心拍変動。心拍の間隔のゆらぎから「緊張しているか／落ち着いているか」を推定できる指標。
SpO2
血中酸素飽和度。血の中に酸素がどれくらい溶けているかの数値。98%以上あれば健康。
SoC
System on Chip。CPUやメモリコントローラや通信機能を1つのチップに詰めたもの。スマホの心臓部。
NPU
Neural Processing Unit。AI計算に特化した回路。CPUより省電力で速くAIを動かせる。
SBC
Single Board Computer。名刺サイズの小型コンピュータ。Raspberry Piが有名。
SQLite
小さなデータベース。1つのファイルにデータを蓄える仕組み。ウェブブラウザ内部でも使われている。
SOUL
Ghost-Printer独自のデータ形式。AIがあなたを「理解」できるよう情報を整えた構造化フォーマット。
Bonsai 1.7B
Ghost-Printer内部で動く小型LLM（17億パラメータ・1bit量子化・240MB）。役割は意味抽出と重要度判定。会話はしない — その結果をSOULに統合し、実際の返答は外部AI（Claude等）が生成する。
Pogo Pin
スプリング付きの金属ピン。押し付けるだけで電気的接続ができる。磁石と組み合わせて充電に使う。
BlueZ
Linux標準のBluetoothソフトウェア。ハードとアプリの橋渡しをする。
