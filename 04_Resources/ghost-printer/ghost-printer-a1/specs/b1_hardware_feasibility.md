# Ghost-Printer B1 — ハードウェア実現性レポート

**作成日:** 2026-04-15
**対象:** Pi Zero 2W (Cortex-A53 1GHz, 512MB RAM) でのAIモデルスタック動作可否

---

## 結論

**Pi Zero 2Wでの動作は技術的に可能だが、512MB RAMが最大のボトルネック。**
3モデル同時ロードは不可能で、Tier別の順次ロード（排他実行）が必須。
製品版では **Pi 5 (2GB)** への移行を推奨する。

---

## 各モデルの実現性

### Whisper tiny (39MB) — 音声→テキスト
- **判定: 動作可能**
- 推論速度: 実時間の2-4倍（1分の音声処理に2-4分）
- RAMフットプリント: 約55MB（モデル39MB + ランタイム）
- whisper.cpp (C++) が推奨実装
- Pi 5なら6秒/10秒音声（約12倍高速）

### 小型LLM 1-2Bパラメータ — テキスト分析
- **判定: 制約付きで動作可能**
- 推奨モデル: Qwen2.5 0.5B (INT8量子化, ~150MB) または TinyLlama 1.1B (INT8, ~180MB)
- 推論速度: 8-15 tokens/sec (Pi Zero 2W) vs 40 t/s (Pi 5)
- メモリマッピング技術(PicoLM方式)で30MB作業メモリで1Bモデル実行可能
- 日本語対応: TinySwallow-1.5B-Instruct が日本語感情分析で検証済み

### all-MiniLM-L6-v2 (22MB) — 埋め込みベクトル
- **判定: 容易に動作可能**
- 最も軽量。推論300-500ms/文
- ONNX/TFLite形式で最適化可能

---

## Tier別実行パイプライン（推奨アーキテクチャ）

```
[音声入力] → [Whisper tiny ロード: 55MB] → テキスト化 (2-4分)
          → [Whisper アンロード]
          → [LLM ロード: 180MB] → 性格/感情抽出 (7-15秒)
          → [LLM アンロード]
          → [MiniLM ロード: 50MB] → ベクトル生成 (0.3秒)
          → [MiniLM アンロード]
          → [SOUL更新] → 完了
```

**パイプライン合計: 10-20秒/入力（音声処理を除く）**
